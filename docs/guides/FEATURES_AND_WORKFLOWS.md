# Features and Workflows

## Product Features

- Dashboard with training and health visibility
- Garmin activity sync
- Workout ingestion and analysis
- AI-assisted training plan generation and adaptation
- Race catalog search and filtering

## Key User Flows

### New User Flow
1. Register / log in
2. Complete profile and goals
3. Connect Garmin (optional but recommended)
4. Generate first training plan

### Ongoing Athlete Flow
1. Sync latest activities
2. Review dashboard and trends
3. Track plan adherence
4. Adjust plan based on feedback/performance

## Data Clarification

- **Race catalog** (seeded events) is not the same as
- **User workout history** (synced/uploaded personal activities)

These datasets serve different product functions and should be interpreted separately.
